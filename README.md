# ETF套利分析工具

一个专业的ETF溢价率套利分析工具，支持数据采集、跟踪误差分析、可视化报告生成。

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install akshare pandas numpy plotly

# 2. 配置ETF代码（编辑 config.json）
{
  "etf_codes": ["513100", "513390"],
  "benchmark": "纳斯达克"
}

# 3. 运行分析
python etf_arbitrage.py

# 4. 查看报告（用浏览器打开）
start data/report.html        # Windows
open data/report.html         # Mac
xdg-open data/report.html     # Linux
```

## ✨ 核心功能

- **数据采集**: 自动获取ETF价格、净值和指数数据
- **智能缓存**: 避免重复请求，支持增量更新
- **溢价率分析**: 计算并可视化ETF溢价率走势
- **跟踪误差分析**: 评估ETF跟踪指数的准确性
- **套利机会识别**: 自动识别ETF间的套利机会
- **交互式报告**: 生成可交互的HTML图表报告

## 📊 支持的图表

1. **溢价率走势图** - 多ETF溢价率时间序列对比
2. **跟踪误差走势图** - ETF相对于指数的跟踪偏离
3. **累计收益对比图** - ETF与指数的归一化收益对比
4. **溢价率差值走势图** - ETF配对间的套利空间分析
5. **溢价率分布图** - 各ETF溢价率分布特征
6. **套利统计表** - 详细的套利机会统计（可排序）

### 关键指标

**溢价率** = (市场价格 - 基金净值) / 基金净值 × 100%

- **正溢价**: 市场价格 > 净值（ETF被高估）
- **负溢价**: 市场价格 < 净值（ETF被低估）

**套利逻辑**:
1. 当ETF A溢价率 - ETF B溢价率 > 2%时
2. 卖出高溢价ETF A，买入低溢价ETF B
3. 当差值回归时平仓获利

## ❓ 常见问题

**Q: 为什么需要获取净值数据？**  
A: 套利公式是：(价格A - 净值A)/净值A - (价格B - 净值B)/净值B。没有净值就无法计算真实溢价率。

**Q: 数据更新频率？**  
A: 数据缓存后不会自动更新。如需最新数据，删除 `data/cache/` 目录重新运行。

**Q: 可以实盘交易吗？**  
A: 本工具仅供分析参考。实盘前建议：
1. 用小资金测试
2. 考虑交易成本
3. 设置止损

**Q: 如何导出数据到Excel？**  
A: 分析结果已保存为CSV格式，可以直接用Excel打开：
```bash
# Windows
start data/513100_premium.csv
```

**Q: 图表可以离线查看吗？**  
A: 可以！生成的HTML文件是独立的，不需要网络连接。
